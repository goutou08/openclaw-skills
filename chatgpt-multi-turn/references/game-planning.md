# 游戏策划系列 Prompt 模板

> 适用于：使用 OpenClaw 在 ChatGPT 上进行游戏策划相关的多轮深度对话

## 第 1 轮：基础认知

**Prompt：**
```
openclaw如何参与游戏策划设计工作
```

**预期输出：**
- OpenClaw 在游戏策划各阶段的能力
- 从创意到落地的完整流程
- AI vs 人类各自的角色定位

---

## 第 2 轮：Prompt 模板获取

**Prompt：**
```
请给我一个具体的OpenClaw游戏策划Prompt模板，让我能直接用于实际项目
```

**预期输出：**
- 总控 Prompt（主策划人格）
- 玩法设计 Prompt 模板
- 使用技巧（追问方式、上下文固定方法）

---

## 第 3 轮：游戏类型定制（示例：SLG）

**Prompt：**
```
我现在要做一款SLG手游，请给我一套专门适配SLG游戏类型的专用Prompt包
```

**Prompt（变体：卡牌）：**
```
我现在要做一款卡牌手游，请给我一套专门适配卡牌游戏类型的专用Prompt包
```

**Prompt（变体：独立游戏）：**
```
我现在要做一款独立游戏，请给我一套专门适配独立游戏类型的专用Prompt包
```

**预期输出：**
- 该游戏类型的核心设计要点
- 专用主策划人格 Prompt
- 该类型常见的坑和注意事项

---

## 第 4 轮：项目实战（可选）

**Prompt：**
```
请针对 [你的游戏概念]，使用上述 SLG Prompt 包，给我设计一个完整的游戏方案
```

**预期输出：**
- Core Loop 设计
- 核心系统设计
- 数值框架
- 商业化设计思路

---

## 使用建议

1. **第1轮**获取通用认知和方法论
2. **第2轮**获取可直接用的 Prompt 模板
3. **第3轮**获取针对你的游戏类型的定制化方案
4. **第4轮**（可选）让 AI 直接参与你的项目设计

---

## 注意事项

- 根据实际项目类型替换第3轮 Prompt 中的"SLG"
- 第4轮需要提供足够的项目背景信息
- AI 输出需要人工审核和判断
